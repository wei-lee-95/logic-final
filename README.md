# 宮心劇場：小主求生指南

一款以宮廷權謀為主題的互動式劇情網頁遊戲。玩家輸入自己的名號後，將以「小主」身分進入後宮，在不同情境中做出選擇，推進故事分支並解鎖不同結局。

## 網頁連結

[https://wei-lee-95.github.io/logic-final/](https://wei-lee-95.github.io/logic-final/)

## 專案特色

- 互動式分岔劇情：每一次選擇都會影響後續發展。
- 10 種不同結局：包含失勢、逆襲、共治、稱霸等多種路線。
- 玩家名稱記錄：可輸入自己的名號，增加遊戲代入感。
- 結局解鎖統計：透過 Firebase Realtime Database 儲存玩家已達成的結局。
- 沉浸式氛圍：搭配宮廷背景圖、古風配色與背景音樂。
- 重新遊玩機制：結局後可再次挑戰，嘗試解鎖其他路線。

## 遊戲玩法

1. 進入網頁後，輸入你的名號。
2. 點擊「開始遊戲」進入劇情。
3. 閱讀故事文字，依照情境選擇行動。
4. 抵達結局後，系統會顯示本次結局與目前解鎖進度。
5. 點擊「再玩一次」可以重新開始，挑戰其他結局。

## 使用技術

- HTML：建立網頁結構與遊戲畫面。
- CSS：設計宮廷風格介面、背景與按鈕樣式。
- JavaScript：控制劇情流程、選項切換、結局顯示與重玩邏輯。
- Firebase Realtime Database：記錄每位玩家已解鎖的結局。
- GitHub Pages：部署靜態網頁。
